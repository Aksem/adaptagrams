Detailed Description

A cluster defines a hierarchical partitioning over the nodes which should be kept disjoint by the layout somehow.

You should not use this directly. This is an abstract base class. At the top level you should be using RootCluster, and then below that either RectangualarCLuster or ConvexCluster.
